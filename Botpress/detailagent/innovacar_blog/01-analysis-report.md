## Bölüm D — Karar Noktaları (Kullanıcı Onayı Gerekli)

1. **G1 GLOSSY (glaze) karşılığı:** Kataloğumuzda saf "glaze" kategorisi yok. Üç opsiyon:
   - **(A)** Blogdaki glaze adımını çıkarıp akışı kısalt (4 aşamaya düşür: yıkama → decon → cila → koruma)
   - **(B)** En yakın ürün olarak W1 QUICK DETAILER (79301) ata (parlatma fonksiyonu yakın)
   - **(C)** Menzerna One-Step Polish (22748.261.001) ata (ama zaten A1 ALL IN ONE için kullanılıyor, çakışır)
   - **Önerim: (A)** → glaze adımı opsiyonel olarak işaretle; ürün atama yapma.

2. **SD1 IRON THIXO RAPID karşılığı:** Bizde demir+şampuan birleşimi yok. Kombinasyon önerisi (D1 IRON + S2 FOAMY ardışık) mantıklı mı?

3. **100% SCALE = 0 SCALE kabul:** Innovacar'ın yeni ürün adlandırmasında 0 SCALE ile 100% SCALE aynı ürün. Katalogta 0 SCALE var → eşdeğer kabul edilebilir. ONAY bekleniyor.

4. **MICRON serisi marka-karışık eşdeğerler:** Mikrofiber için farklı markalar (GYEON / KLIN) öneriliyor. Tek markaya odaklanmak ister misin, yoksa SKU-bazlı en iyi eşdeğer yeterli mi?

5. **SINH TOP COAT vs SINH HYBRID:** İki ürün farklı seviye seramik (TOP COAT = hazırlık gerektiren profesyonel, HYBRID = tek adım). Birebir eşdeğer değil ama aynı kategori. SINH HYBRID + (opsiyonel MX-PRO DIAMOND premium upgrade önerisi) sunulsun mu?
